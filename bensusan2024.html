The pan-mnemist spectral realism I have been elaborating considers the situated present to be no more than a thin layer of confluences between the vulnerability of archives found everywhere and the addends that can come from anywhere. It is a spectral confluence in the sense that there is no complete furniture of the world. This spectral realism ascribes no exceptional status to human minds as far as memory is concerned—human memory is conceived as a memory assemblage of other memory assemblages that are constitutive of animal organisms. Thus remembering, in the usual employment of the word only to apply to human internal memory, is an instance of a large collection of operations of retrieval that are in place as a consequence of being composed of memory assemblages, with dedicated archives and a set of operations to recall what is there (BENSUSAN, 2024, p. 173)
